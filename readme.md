## Pre Conditions

1. Node.js LTS (8.9.4)

## Instalation

Install it as (in this sequence):

    $ npm install
    $ npm run updatewd

## Usage

Too run tests in terminal, type:

```
  npm test
```

## Reports

1. To see reports, open the .hmtl file on firefox on path:

```
./e2e/tests_result/cucumber_report.html

```
